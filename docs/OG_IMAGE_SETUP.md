# OG Image Generation Instructions

## Option 1: Using the HTML Generator (Recommended)
1. Open `generate-og.html` in your browser
2. Click the "Download OG Image" button
3. Save the file as `public/og-image.png`

## Option 2: Using the SVG
1. Open `public/og-image.svg` in a browser or image editor
2. Export/screenshot as 1200x630 PNG
3. Save as `public/og-image.png`

## Option 3: Using an Online Tool
1. Go to https://og-image.vercel.app/
2. Enter:
   - Title: "MC COLLO EVENTS"
   - Subtitle: "Professional Master of Ceremonies"
3. Download and save as `public/og-image.png`

## After generating:
- Place the PNG file in the `public/` folder
- Commit and push to trigger a Vercel deployment
- Test by sharing your URL on social media or using: https://metatags.io/
